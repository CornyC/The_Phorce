<!-- INSTALLATION -->
## Installation
Requires at least Python 3.8. Uses numpy, MDAnalysis, and OpenMM. 

1. Clone the repo
   ```sh
   git clone https://github.com/CornyC/The_Phorce.git
   ``` 
2. Go into the directory
   ```sh
   cd The_Phorce
   ``` 
3. Install
   ```sh
   python setup.py install
   ```
